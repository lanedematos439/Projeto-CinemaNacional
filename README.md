# 🎬 CinemaNacional

Plataforma de Streaming desenvolvido com o objetivo de valorizar o Cinema Brasileiro.

## 📖 Sobre o projeto
O CinemaNacional é uma plataforma de Streaming que uniu duas paixões minha: a programação e o Cinema Brasileiro, fiz com o objetivo de ter a experiência de um projeto com backend
completo e colocar em prática meus conhecimentos em PHP e Banco de Dados.


## Tecnologias utilizadas
- HTML
- CSS
- PHP
- SQL
- JavaScript

## Funcionalidades
- Listagem de filmes brasileiros
- Login e Cadastro
- Página com informações de cada filme
- Interface simples e intuitiva
